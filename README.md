# doong-jo.github.io

## Getting Started

1. Install Live Preview Extension for VS Code
2. Open the project in VS Code
3. Press `Ctrl+Shift+P` to open the Command Palette
4. Type `Live Preview: Show Preview` and press Enter
5. The preview will open in a new browser tab
